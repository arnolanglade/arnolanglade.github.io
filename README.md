# Develop

Install the dependencies for Bookshop:

~~~bash
$ npm install
~~~

Install the Jekyll dependencies with [Bundler](http://bundler.io/):

~~~bash
$ npm run install-jekyll
~~~

Run the website:

~~~bash
$ npm start
~~~
